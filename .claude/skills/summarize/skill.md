# Summarize Content Skill

## When to Use
- Long articles or documents need quick overview
- Extract main points from text
- Create study notes from lectures
- Condense meeting transcripts
- Simplify complex information

## Procedure
1. Read and understand the content
2. Identify main ideas and key points
3. Remove redundancy and details
4. Format as clear bullet points
5. Organize by importance or topic

## Output Format

**Main Points**
• [Key point 1]
• [Key point 2]
• [Key point 3]

**Details** (if needed)
• [Supporting detail 1]
• [Supporting detail 2]

**Action Items** (if applicable)
• [Action 1]
• [Action 2]
