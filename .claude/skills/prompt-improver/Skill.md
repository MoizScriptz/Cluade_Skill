# Prompt Improver Skill

## When to Use
- Vague prompts need clarity
- Want better AI responses
- Need structured output

## Procedure
1. Analyze original prompt
2. Add missing elements
3. Present improved version

## Output Format

**Original Prompt**
[User's prompt]

**Improved Prompt**
[Enhanced version]

**Changes Made**
- [What was added]

## Prompt Formula

**Role + Task + Format + Audience + Tone**

## Examples

**Bad:**
> "Write about dogs"

**Good:**
> "Write a 300-word article about dog exercise benefits for pet owners. Friendly tone, include 3 benefits."

---

**Bad:**
> "Code help"

**Good:**
> "Write Python function to validate emails. Include type hints and 3 test cases."
